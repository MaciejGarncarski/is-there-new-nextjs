# Project

Is there new Next.js?

Unofficial Next.js release tracker.

Live demo: https://is-there-new-nextjs.vercel.app

## Quick start

Prerequisites: Node.js and pnpm.

Install and run locally:

```sh
pnpm install
pnpm run dev
```

## Tech used

- Next.js (framework)
- TypeScript
- Node.js
- pnpm (package manager)
- ESLint
